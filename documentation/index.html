Converge Payment Gateway Plugin for WooCommerce

Elavon's Converge Payment Gateway

The Converge Payment Gateway plugin for WooCommerce enables you to take credit card payments directly from your WooCommerce site using a Converge account (formerly Virtual Merchant) provided by Elavon.

System Requirements

The Elavon/Converge Payment Gateway plugin for WooCommerce requires the following setup:

  1. An installation of Wordpress 3.8 or above. Wordpress may be downloaded from http://wordpress.org/.
  2. WooCommerce 2.1.0 or later. WooCommerce may be downloaded from http://www.woothemes.com/woocommerce/.
  3. A Converge account (formerly Virtual Merchant) from Elavon. Go to http://www.elavon.com/ for more information.

To use this plugin, you must have the force secure checkout option enabled within WooCommerce, and a valid SSL certificate (all of which is required prior to Elavon/Converge's initial account approval process).

Installation

 

  1. From your Wordpress admin dashboard, select "Plugins"
  2. Select the "Add New" link followed by the "Upload" link
  3. Select the "Choose File" button
  4. Browse your computer to the downloaded "woocommerce-gateway-elavon-converge.zip" file and select "Open"
  5. Select "Install Now"
  6. Click on the "Activate Plugin" link, and the plugin is installed!
 

Configuration

To configure the plugin, select "WooCommerce" from your admin dashboard and follow these steps:

  1. Click on the "Settings" link
  2. Select the "Checkout" tab
  3. Click on the "Converge" link located at the top of the page
  4. WARNING: Verify that the currency setting for WooCommerce matches the currency setting for your Converge account.
  5. Check the box to enable the Converge Payment Gateway.
  6. Enter the title you wish to be displayed for this payment option on the Checkout page of your site. The default is "Credit Card/Debit Card".
  7. Provide a description to be displayed below the title on the Checkout page.
  8. If you wish to test the plugin, enable the "Converge Test" option. A unique testing account from Converge is required for the testing option to work. Contact Converge at 1-800-377-3962 to request a unique testing account.
  9. Provide the Merchant ID, User ID and pin provided by Elavon/Converge for your account. Please note, these will be different from the Account ID, User ID, and Password used to log in to your terminal
  10. If you wish to accept purchasing cards, select the box to enable the use of purchasing cards.
  11. Enable authorization only purchases by selecting the "Enable authorization only purchases." box. This type of transaction creates an authorization with the card issuing bank that holds the funds until you wish to manually capture them.
  12. Select "Enable CSC Authentication" if you would like to test the card security code for validation as a condition of sale approval.
  13. Select a level for AVS Authentication. The address verification system helps to prevent credit card fraud by confirming the address records of the credit card holder.
  14. Select "Save Changes" to save all changes made to your plugin configuration.

Configuration for the plugin is now complete.
